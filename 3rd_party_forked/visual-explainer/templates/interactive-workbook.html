Interactive workbook template

Capture structured answers directly inside the page.

Use this template when the page is not just an explanation, but also a tool for collecting decisions, approvals, review notes, or questionnaire answers. Mix question types as needed — not every prompt needs A/B/C.

Design rule

Choose the answer model per prompt: single-select, multi-select, text-only, or mixed preset + notes. The content should decide the structure — not the other way around.

Export rule

If the user will answer multiple prompts, include a bottom summary/export panel. If choices and notes can both exist, the summary should combine them.
