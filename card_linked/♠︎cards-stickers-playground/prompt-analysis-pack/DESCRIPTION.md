This folder contains potentially useful prompts for LLM agents focused on prompt evaluation, scoring, and iterative improvement.

The pack is designed to turn vague prompt review into structured, machine-readable feedback by checking for actionability, clarity, behavioral constraints, drift prevention, and signal-to-noise.

In practical terms, this is a compact prompt-QA toolkit: one prompt scores and critiques a prompt, while the other scores it and produces an improved rewrite. It is most useful when you want prompt review to be more systematic, comparable, and automation-friendly.
